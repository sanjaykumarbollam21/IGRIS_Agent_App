# Pinned for reproducible "Hey IGRIS" wake word training.
# Tested on Python 3.11 (your venv) and 3.12.
#
# Install with (PowerShell):
#     python -m venv .venv-wakeword
#     .\.venv-wakeword\Scripts\Activate.ps1
#     python -m pip install --upgrade pip
#     pip install -r tools/requirements-wakeword.txt
#
# Note on Python 3.13: tensorflow 2.17+ adds cp313 wheels; we pin 2.18.0
# which works on 3.9-3.12. If you're on 3.13 and hit "no wheels", drop
# the version back to 2.17.0 (first TF with cp313) or 2.18.0 (current).
#
# We also pin the CPU-only PyTorch wheels (multi-GB CUDA wheels are
# otherwise pulled by default on Windows). The `--extra-index-url`
# directive at the TOP of the file tells pip where to find them.

--extra-index-url https://download.pytorch.org/whl/cpu
